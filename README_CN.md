# 🎉 Feature List

- [x] 支持通过 URL / 插件 / Telegram 进行网页收藏
- [x] 支持对收藏内容划线、评论、回复、分享、加星标、归档
- [x] 支持对收藏进行AI对话、AI总结、划线对话、AI标签生成
- [x] 支持对收藏内容进行关键词或语义搜索
- [x] 支持从omnivore导入收藏
- [x] 支持 Websocket / Browser Push 进行消息推送
- [x] 支持服务端渲染及 Web app 使用

# Slax Reader 账户注销须知

为保证你的账号安全，在你提交的注销申请生效前，需同时满足以下条件：

## 1. 注销需先行通过账号安全验证

请确保你能正常登录账号，并完成身份验证流程。

## 2. 账号财产已结清，订阅关系已处理

### 2.1 订阅服务相关

- **无正在进行的订阅**：如果您有正在进行的订阅，请先取消订阅后再试
- **无活跃的合集订阅**：如果您订阅了某些合集，请先取消这些合集的订阅
- **无免费订阅历史问题**：如果您有免费订阅记录导致无法注销，请联系客服处理

### 2.2 创作者账号相关

- **合集无订阅者**：如果您是创作者，且您的合集有订阅者，请先处理相关合集（转让、删除或等待订阅期结束）
- **Stripe Connect 已解除**：如果您的账号关联了 Stripe Connect 支付账户，请先解除关联

账号下所有关联业务的资产、预期收益和权益均已结清、退款、清空或自愿放弃，所有交易已完成或已自愿放弃。

## 3. 账号授权、绑定已解除

账号已解除与第三方产品、平台的授权或绑定关系（如 Google、Apple 等社交账号登录），通过本账号接入的相关服务中没有未完成或存在争议的内容。

---

## 常见注销失败原因

如果您在注销账户时遇到错误，请根据以下提示进行处理：

| 失败原因                      | 解决方案                                                                   |
| ----------------------------- | -------------------------------------------------------------------------- |
| 您有正在进行的订阅            | 前往"我的订阅"页面，取消所有正在进行的订阅，等待订阅期结束后再试           |
| 您订阅了某些合集              | 前往"我的订阅"页面，取消所有合集订阅后再试                                 |
| 您的合集有订阅者              | 作为创作者，请先处理您的合集：可以删除合集、转让所有权或等待所有订阅期结束 |
| 您的账号关联了 Stripe Connect | 前往"创作者设置"页面，解除 Stripe Connect 关联后再试                       |
| 您有免费订阅记录              | 请联系 Slax Reader 客服团队，我们将协助您处理                              |

## 重要提示

- 账户注销后，您的所有数据将被永久删除且无法恢复，且30天内无法重新注册相同账号
- 注销前请确保已备份您需要保留的内容
- 如有任何疑问，请联系客服支持

## 我们收集的信息

本声明适用于在 Slax Reader 网站以及 Slax Reader 应用程序和服务（包括 iPhone、iPad、Android、Kindle 及其他设备和平台）上收集或提交的所有信息，以下简称 Slax Reader。

Slax Reader 偶尔需要出于便利性、实用性或技术要求收集信息。这些信息包括：

- 您选择的用户名，可能是您的电子邮件地址或姓名。从技术上讲，用户名是必需的，这样您就可以从多台计算机和设备登录。我们鼓励您使用电子邮件地址以实现"忘记密码"功能。
- 您从网站、我们的应用程序、使用我们 API 的第三方应用程序以及任何其他方式保存到 Slax Reader 的页面标题和元数据。
- 当识别为需要登录的已知站点时，您保存到 Slax Reader 的页面的 HTML 内容。这在技术上是生成文本视图和其他索引功能所必需的，适用于需要登录或账户的网站，例如一些主要的新闻出版商。当为页面保存 HTML 时，它仅在您的账户内使用，不会通过社交或共享功能显示。

我们还可能使用"cookie"技术。除其他事项外，我们可能使用 cookie 或其他跟踪技术来帮助您在页面之间高效导航，使您无需每次访问时重新输入该信息。
